# New York Times Article Scraper

## About this Project

This M.E.R.N app lets you search and scrape the New York Times article database. 

## To run app

Run 'yarn install' to deploy app locally.

## Technologies Used

- node.js
- express
- mongoDB
- REACT.js
- Materialize
- mongoose
- axios
